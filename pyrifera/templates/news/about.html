Welcome to Pyrifera

Pyrifera is an open-source application for visualizing the results of ecological monitoring built by the McClintock lab at the University of California Santa Barbara. The source code is freely available on the project code site

Pyrifera is built on top of MarineMap, an open-source framework for the development of decision support tools for Marine Spatial Planning. MarineMap was developed by collaborators at the University of California Santa Barbara, Ecotrust, and The Nature Conservancy. Read more at marinemap.org
